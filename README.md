# Информационная система "БАДы"
Информационная система состоит из:
- Базы данных, содержащей наименования БАД; 
- интерфейса, который позволяет оператору со стороны Заказчика быстро и удобно искать, добавлять и удалять наименования БАДов в БД;
- бэкенд-части с доступом по API, позволяющей подключать ИС к другим продуктам;
- модуля резервного копирования на Яндекс Диск.
